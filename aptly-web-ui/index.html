Goals

This Web Interface aims to provide a useful visualization and administration toolset to interact directly on Aptly's REST interface.

Any improvements to data representation and/or new Routes additions should be made in Aptly's home repository

With this out of the way, here is the projected feature set :

  • Repositories and packages listing (OK)
  • Per-Repo Package version and details (OK)
  • Packages Upload and deletion (OK)
  • Package migration (WIP)
  • Snapshot creation and management

The UI is currently usable but with a lot of missing features and some rough edges. Don't expect every error to be handled cleanly !
